A new WebFX demo!

Sep 25, 2022

You may haven’t noticed it, because we are writing this post only now, but we published a new demo on the WebFX website about a month ago.

This demo is based on the excellent DemoFX library by Chris Newman. It’s a JavaFX canvas library that offers a collection of effects which were primarily designed for JavaFX canvas benchmarks, but can be used and chained to make animations too. We made a partial port of this library in WebFX Libs, our dedicated space for WebFX libraries.

We first made this DemoFX kitchen sink to test and visualize each effect individually, and finally we chose to sequence them in a particular order, with the music of David Newman played in the background (also provided by the DemoFX library). We selected specific transition times to match the musical rhythm. The demo starts with an introduction animation, asking to click to play the actual demo, because in the browser sandbox, it’s not possible to start the music without a user interaction first. This resulted in this single java source file.

Most animations should be fluid, except maybe the fractal rings (2nd in the sequence) on low devices or big screens, especially when another animation is running simultaneously (such as the fading star field at the beginning or the Sierpinski effect at the end). You may try the native version of the demo to get better results. But on modern devices (we tested on an iPad mini 6), the performance is excellent already with the web version.

This WebFX demo mainly emphasis the canvas support and the MediaPlayer sound support, including the audio spectrum analyse as demonstrated by the equaliser. WebFX supports the PixelWriter API, but also provides an alternate API, slightly different, which provides better performance for the web (we will make a separate post on this topic later). So we rewrote some DemoFX effects using that alternate PixelWriter API to ensure the best performances. It also shows how WebFX can produce lightweight and performant web apps (the JS size is 128kB, so it loads and starts quickly).
